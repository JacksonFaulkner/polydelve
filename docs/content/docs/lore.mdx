## Origin

Started as **Action Odds** — a prediction market for company-level security events (breaches, phishing, source code leaks). Graded companies A–F and let users bet on incidents.

Pivoted to **Polydelve** when package-level vulnerability data (CVEs, EPSS, MAL advisories) turned out to be richer and more tractable than company-level incident prediction.

## Compliance

Proud to be the only company to fail a SOC 2 audit by Delve.

![NOT SOC 2 Compliant](https://www.aomni.com/landing/assets/soc2.png)

## Name

"Polydelve" = Polymarket + Delve. Delve being the security firm that audited us out of existence.

## Stack evolution

Originally built against a different data model (Markets, Bets, Companies). Those legacy models are still in `models.py` and slowly getting cleaned up.

## Frontend inspiration

UI patterns borrowed from [PolymarketDevs](https://x.com/PolymarketDevs). This implementation is adapted for software-security forecasting.
